# 🦏 Creative Habits
👋 안녕하세요 Frontend Developer 윤태연입니다.  <br />
🧔 진로에 한계를 얻고 방황하던 차 아버지께 개발을 접하고 내가 생각한대로 기능 개발하는 것에 매료돼 개발을 시작했습니다. 그러면서 과정과 문제 해결에 희열감을 느껴 프로그래밍의 길을 선택하게 됐습니다. <br />
